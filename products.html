Mimic

Coming Soon

Project Summary: Mimic is a high-performance, serverless Chrome extension for automated end-to-end testing. It uses the Chrome Debugger API to simulate authentic hardware events such as file uploads and trusted clicks that typical scripts cannot trigger.

Core Components: Capture Engine records precise user interactions and builds robust selectors; Storage Layer uses IndexedDB for high-volume local data including event flows and file blobs; Replay Controller runs recorded sequences with realistic timing and visual playback feedback.

Planned release: 2026

Shift

Coming Soon

Project Summary: Shift is a high-performance, minimalist 2D vertical racer built as a serverless Chrome Extension in the Side Panel. It emphasizes precision gameplay, fast retries, and zero-cost infrastructure.

Technical Core: Custom Canvas 2D engine targeting 60 FPS; Manifest V3 + Side Panel API; seeded PRNG for a deterministic Track of the Day; lightweight AABB collision detection for car-to-wall physics.

Full-Stack Edge (Ghost Mode): Instead of video, Shift records telemetry (position and speed) in chrome.storage.local and replays high-score runs as a transparent ghost car using linear interpolation (lerp) for smooth motion.

Planned release: 2026

Domino

Coming Soon

Project: Domino. Category: Physics-Based Arcade / Chrome Extension. Domino is a high-octane 3D line-clear puzzler that transforms any website into a gravity-bound game board by treating real DOM elements (buttons, images, headers) as physical platforms.

Core Gameplay: 3D voxel blocks drop from the top of the browser window, land and bounce across the page content, and challenge players to fill every horizontal cell in a row.

The Blast: Completing a row triggers a physics-based explosion, clears the line, and causes the blocks above to cascade downward.

Planned release: 2026

Snappy

Coming Soon

Snappy is a lightweight, cross-platform utility that lives in your system tray (Windows) or menu bar (macOS/Linux). It offers a high-speed, "one-click" solution for capturing your entire desktop—across all connected displays—instantly.

  • One-Click Capture: Take screenshots of every monitor by clicking the tray/menu icon.
  • Multi-Monitor Support: Automatically detects all connected screens and saves individual files (Screen_1.png, Screen_2.png, etc.).
  • Custom Watermarking: Add optional text or image overlays, fully configurable in the settings.
  • System Integration: Silent, minimal resource usage; runs pleasantly in the background.
